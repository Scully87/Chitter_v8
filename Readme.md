###Creating a Twitter clone in Ruby

####Technologies Used:

  - Ruby
  - Sinatra
  - RSpec
  - JQuery
  - PostgresQL
  - BCrypt
  - Rack-Flash
  - Capybara

How to use
----------
Clone the repository:
```shell
$ git clone git@github.com:scully87/Bookmark_Manager.git
```

Change into the directory:
```shell
$ cd Bookmark_Manager
```

Open in command line:
```shell
$ rackup
in browser http://localhost:9292
```
